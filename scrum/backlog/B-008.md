Move template governance files into the Scrum memory root so `CONSTITUTION.md` and LLM-facing instructions live under `scrum/` in the default template package.

## Scope

```yaml
section: backlog.item.B-008.scope

  - Move `templates/default/CONSTITUTION.md.j2` to `templates/default/scrum/CONSTITUTION.md.j2`.
  - Move `templates/default/instructions/LLM.md` to a path under `templates/default/scrum/`.
  - Update `templates/default/manifest.yaml` file and instruction mappings.
  - Update any affected package references, render targets, tests, and documentation.

  - Adding root-level proxy files unless explicitly approved.
```
