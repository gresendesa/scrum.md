## Risk

```yaml
section: sprint.SPR-2026-06.risk
title: Risk
status: active
owner: Guilherme
classification: medium
reason: `smd init` writes project memory and configuration into a target root, so overwrite protection and package verification must be reliable.
mitigations:
  - Reuse the checksum-only package contract delivered by B-006.
  - Refuse existing Scrum memory unless an explicit overwrite option is provided.
  - Keep web download, remote trust, cache, and interactive wizard behavior out of scope.
tags: [risk]
```

## Guardrails

```yaml
section: sprint.SPR-2026-06.guardrails
title: Guardrails
status: active
owner: Guilherme
tags: [guardrails]
```

* Do not implement remote template downloads.
* Do not implement remote trust registry, key server, or cryptographic signing.
* Do not move template governance files in this sprint.
* Do not create an interactive TUI or wizard.
* Keep output deterministic and compatible with the stable JSON envelope.
