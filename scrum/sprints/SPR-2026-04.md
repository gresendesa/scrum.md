## Guardrails

```yaml
section: sprint.SPR-2026-04.guardrails
title: Guardrails
status: active
owner: Guilherme
tags: [guardrails]
```

* Do not implement full backlog, sprint, gate, or context command behavior.
* Do not implement the full `smd init` workflow in this sprint.
* Do not implement web download of template packages in this sprint.
* Do not implement trust, signature, checksum, cache, or version policy for template packages in this sprint.
* Do not expose MDBind directly as the normal agent workflow.
* Keep `smd` command outputs deterministic and JSON-friendly.
* B-005 remains deferred until this sprint provides stable examples.
